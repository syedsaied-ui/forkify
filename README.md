# 🍴 Forkify Project

A recipe application that allows users to search, bookmark, and upload recipes.

## 🚀 Features
- Search recipes
- View detailed recipe information
- Bookmark recipes
- Upload custom recipes

## 🛠️ Technologies Used
- JavaScript (ES6+)
- HTML
- CSS
- Parcel (Bundler)
- API Integration

## 📦 Installation & Setup

1. Clone the repository:
git clone https://github.com/syedsaied-ui/forkify.git

2. Navigate into the project folder:
cd forkify

3. Install dependencies:
npm install

4. Start development server:
npm start

## ▶️ How to Run

After running the above commands, open your browser and go to:
http://localhost:1234
